# 관리자 공고 초안 편집 개선

## 날짜

2026-07-24

## 작업 목표

- Codex AI 생성 초안의 제목을 관리자가 직접 수정할 수 있게 합니다.
- 학교 공고 선택과 초안 생성 흐름을 더 빠르게 만듭니다.

## 변경 사항

- 생성 초안의 첫 항목에 공고 제목 입력칸을 추가하고 수정값이 공개 공고 제목에 반영되게 했습니다.
- 요약, FAQ, 답변 근거 입력 영역과 초안 패널의 크기를 확대했습니다.
- 학교 홈페이지 공고를 한 번 클릭하면 선택하고 두 번 클릭하면 즉시 초안을 생성하도록 추가했습니다.
- 기존 초안 생성 버튼은 모바일과 키보드 사용자를 위해 그대로 유지했습니다.
- 관리자 검수 체크 영역에서 수정 버튼을 제거하고 공개 승인과 보류 버튼만 유지했습니다.
- 웹과 Tauri 앱에 동일한 화면과 동작을 반영했습니다.

## 검증

- 웹 `npm test`: 18개 테스트 통과
- 웹 `npm run lint`: 통과
- Tauri `npm run test:preview`: 통과
- Tauri `npm run build`: 통과
- 제목 수정값의 학생 공고 목록 반영 확인
- 기존 버튼 생성과 더블클릭 생성 동시 확인
- 375px 모바일 폭에서 가로 넘침 없음 확인

- 실제 학교 홈페이지 연동은 현재 프로토타입 범위에 포함되지 않습니다.
