# 공고 색상 및 관리자 작업 화면 개선

## 날짜

2026-07-24

## 작업 목표

- 공고 상세 화면에 지정된 녹색 계열 색상 팔레트를 적용합니다.
- 관리자 하위 화면의 중복 로그인 안내를 제거하고 실제 작업 영역을 넓힙니다.

## 변경 사항

- 공고 상세 화면에 `#DAD7CD`, `#A3B18A`, `#588157`, `#3A5A40`, `#344E41` 색상 팔레트를 적용했습니다.
- 관리자 관리, AI 공고 생성 및 공개, 공개 공고 관리 화면에서 중복된 권한 상태 및 학교 계정 로그인 패널을 제거했습니다.
- 관리자 목록은 전체 폭으로, 공고 생성 및 관리 화면은 넓은 2열 작업 공간으로 조정했습니다.
- 웹과 Tauri 앱의 관리자 화면을 동일하게 반영했습니다.
- 관리자 하위 화면에 중복 로그인 패널이 다시 생기지 않도록 통합 테스트를 보강했습니다.

## 검증

- 웹 `npm test`: 18개 테스트 통과
- 웹 `npm run lint`: 통과
- Tauri `npm run test:preview`: 통과
- Tauri `npm run build`: 통과
- 390px 모바일 폭에서 세 관리자 화면의 1열 배치와 가로 넘침 없음 확인
- 웹 `npm run build`: 로컬 전용 `.env.local`이 없어 Firebase 설정 생성 단계에서 중단

## 배포

- GitHub 풀 리퀘스트로 제출하며 Firebase Hosting에는 아직 배포하지 않습니다.

## 남은 이슈

- 실제 Firebase 배포 환경에서 웹 빌드와 최종 화면을 재확인해야 합니다.
