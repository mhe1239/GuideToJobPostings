# 3) 서비스 기획안

## 서비스 개요

강남대 공고 안내 서비스는 학생이 학교 공고를 더 빠르게 이해하고, 필요한 세부 내용을 서비스 안에서 확인할 수 있게 돕는 웹 및 데스크톱 앱입니다.

## 대상 사용자

- 강남대학교 학생
  - 공고 목록을 빠르게 훑고 자신에게 맞는 공고를 찾고 싶음
  - 신청 기간, 지원 대상, 모집 분야, 제출 서류를 먼저 확인하고 싶음
  - 필요할 때 전체 공고 원문과 공식 출처를 확인하고 싶음
- 관리자/운영자
  - 학교 공고를 등록, 검수, 공개하고 싶음
  - 관리자 구성원을 관리하고 싶음

## 제품 구성

### 학생용 웹/앱

- 공고 목록
- 마감일이 가까운 공고 우선 정렬 및 마감 공고 후순위 표시
- 학생 조건 기반 맞춤 필터
- 공고 상세
- 핵심 정보 카드
- FAQ 열기/닫기
- 질문 입력과 답변 표시
- 전체 공고 내용 토글
- 공식 원문 링크
- 출처 및 담당 부서 안내

### 관리자용 웹/앱

- 관리자 로그인
- 공고 초안 생성
- 공고 검수 및 공개
- 공개 공고 관리
- 관리자 멤버 관리

### 배포/패키지

- Firebase Hosting 웹 배포
- Firestore Rules/Indexes
- Cloudflare Worker/D1 API
- Tauri Windows/macOS 설치 패키지

## 핵심 사용자 흐름

1. 학생이 공고 목록에 진입합니다.
2. 관심 분야, 모집 상태, 학생 조건으로 공고를 좁힙니다.
3. 상세 페이지에서 제목, 모집 상태, 신청 기간, 지원 대상, 모집 분야를 확인합니다.
4. FAQ를 열어 자주 묻는 질문을 확인합니다.
5. 직접 질문을 입력해 저장된 공고 기반 답변을 확인합니다.
6. 더 자세한 내용이 필요하면 전체 공고 내용을 펼칩니다.
7. 정확한 확인이 필요하면 공식 원문 링크를 새 탭에서 엽니다.

## 현재 기능 범위

- 공고 목록과 상세 이동
- 모바일/데스크톱 반응형 레이아웃
- FAQ 기반 답변
- 로컬 fallback 답변
- 전체 공고 문서형 렌더링
- 원문 이미지 표시와 이미지 오류 상태
- 공식 원문 링크
- 학생 프로필 기반 맞춤 필터
- 관리자 로그인/멤버/공고 관리
- Firebase Hosting 배포
- Tauri 설치 파일 빌드

## 구현하지 않는 범위

현재 범위에서는 아래 기능을 기본 구현 대상으로 보지 않습니다.

- 실제 OCR
- 학교 홈페이지 실시간 크롤링
- AI 원문 자동 재작성
- PDF 뷰어
- 이미지 확대 모달
- 새로운 로그인/권한 체계
- 공고 내용 자동 번역
- 새 UI 라이브러리 도입

## 운영 원칙

- 학생은 요약 정보를 먼저 보고, 긴 원문은 필요할 때 펼칩니다.
- 요약과 원문은 시각적으로 구분합니다.
- 원문 내용은 의미를 바꾸지 않고 읽기 좋게 표시합니다.
- 공식 출처 확인 경로는 가능한 한 유지합니다.
- 답변 기능이 로컬 fallback으로 동작하더라도 다른 화면 오류와 섞어 판단하지 않습니다.
