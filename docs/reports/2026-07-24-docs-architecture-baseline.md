## 날짜

2026-07-24

## 작업 목표

Austin Docs Architecture의 한국어 문서 운영 방식을 현재 강남대 공고 안내 프로젝트에 맞게 적용합니다.

## 반영한 기준

- 현재 문서는 삭제하지 않고 유지합니다.
- 루트 `AGENTS.md`와 `docs/`를 추가해 AI 에이전트 작업 기준을 둡니다.
- 프로젝트를 “강남대 공고 안내 웹 + Tauri 데스크톱 패키지 + Firebase/Worker 배포” 기준으로 문서화합니다.
- TODO는 다음 작업부터 `docs/todo/`에 남기는 방식으로 운영합니다.
