# 공고 마감일 정렬 및 관리 선택 UI 정리

## 날짜

2026-07-24

## 변경 사항

- 학생 공고 목록을 마감일이 가까운 진행 공고부터 표시하도록 정렬했습니다.
- `facts.period`에 여러 날짜가 있으면 마지막 날짜를 마감일로 사용합니다.
- 마감일을 알 수 없는 진행 공고는 날짜가 확인된 진행 공고 다음에 표시합니다.
- 상태가 `마감`이거나 마감일이 지난 공고는 목록 마지막에 표시합니다.
- `manage.html` 공개 공고 목록에서 `선택 해제` 버튼을 제거했습니다.
- 전체 선택 체크박스를 다시 해제하거나 각 공고 체크박스를 해제하는 기존 동작은 유지합니다.

## 검증

- 웹 `npm test`: 25개 테스트 통과
- 웹 `npm run lint`: 통과
- 기간의 마지막 날짜 추출, 연말·연초 기간, 마감 공고 후순위 정렬 테스트 통과
- 로컬 학생 목록에서 `마감` 공고가 마지막에 표시되는지 확인

## 제외 범위

- Tauri 앱 및 macOS/Windows 설치파일은 수정하거나 검사하지 않았습니다.
- Firebase Hosting에는 아직 배포하지 않았습니다.
