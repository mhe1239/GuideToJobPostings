# 사용자 노출 개발 문구 제거

- 작업 브랜치: `codex/bulk-selection-actions`
- 대상: `publish.html`, `notice.html`, 공통 상세/발행 스크립트 및 Tauri 동기화 소스

## 변경 내용

- 공고 상세 화면에서 프로토타입/가상 샘플 안내 문구를 표시하지 않도록 정리했습니다.
- 상세 화면의 부서 피드백용 내부 검증 섹션을 제거했습니다.
- 공고 선택/상세/내 정보 푸터와 메타 설명에서 프로토타입 표현을 서비스 표현으로 교체했습니다.
- 발행 화면의 학교 홈페이지 공고 선택 안내에서 시뮬레이션, 로컬 JSON, 예시 데이터 문구를 제거했습니다.
- 관리자용 오류 시뮬레이션 버튼을 사용자 화면에서 제거했습니다.
- 미리보기 통합 테스트를 새 문구 정책에 맞게 갱신했습니다.

## 검증 결과

- `01_WEB_FIREBASE`: `npm run lint`, `npm test`, `npm run build` 통과
- `02_TAURI_DEVELOPMENT_SOURCE`: `npm run test:preview`, `npm run build` 통과
- 앱 소스 금지 문구 검색: 프로토타입/가상/샘플/시뮬레이션/피드백용 문구 없음
- 민감정보 검색: 하드코딩된 키 없음, Worker 런타임 secret 참조만 확인
